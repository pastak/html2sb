# html2sb

Encode HTML to Scrapbox.io style text

## Usage

### CLI

- `% npm install -g html2sb`
- `% html2sb hoge.html > hoge.txt`
  - You can pass filename as option
- `% cat hoge.html | html2sb > hoge.txt`
  - You can pass markdown text via stdin
